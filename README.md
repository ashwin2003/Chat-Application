## Installation and Setup Instructions

#### Example:  

Clone down this repository. You will need `node` and `npm` installed globally on your machine.  

Installation:

`npm install`   

To Start Server:

`cd .\server\`
`npm run devStart`  

To start the Application:

 `cd .\client\`
 `npm start`

To Visit App:

For user 1
`localhost:3000`

For User 2 
`localhost:3000` in private mode.
